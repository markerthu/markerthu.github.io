TL;DR ‚Äî Population-based RL methods improve exploration by running diverse policies, but are fundamentally limited by a fixed, predefined population. LBC breaks this limitation by learning a hybrid behavior mapping over all policies, enabling a dramatically enlarged behavior space ‚Äî and achieves superhuman performance with 500√ó less data.

üß© The Core Idea

Population-based methods fix a set of exploratory policies and select between them. LBC instead constructs a continuous, learnable behavior mapping space that blends all policies, then uses a bandit-based meta-controller to learn which behaviors to select at each moment:

üìê Hybrid Behavior Mapping

Instead of selecting from a fixed population, LBC parameterizes a convex combination space over all policies ‚Äî infinite diversity from a finite set of base agents.

üé∞ Bandit Meta-Controller

A lightweight bandit algorithm learns which behavior mapping to activate for each episode, balancing exploration across the behavior space with exploitation of known good behaviors.

üîó Off-Policy Integration

LBC is integrated into distributed off-policy actor-critic methods ‚Äî compatible with existing RL infrastructure without major architectural changes.

üìä Unified Perspective

Provides a unified view of diverse RL algorithms as special cases of behavior control, opening new directions for understanding exploration in deep RL.
